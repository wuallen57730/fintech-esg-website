# ⚡ 快速開始指南

## 🎯 三步驟開始使用

### 第 1 步：設置 API Key（5 分鐘）

1. **獲取 OpenAI API Key**
   - 訪問：https://platform.openai.com/api-keys
   - 登入並點擊 "Create new secret key"
   - 複製您的 API Key（格式：`sk-proj-xxxxx...`）

2. **配置系統**
   - 打開 `config.js` 文件
   - 找到第 5 行：`OPENAI_API_KEY: 'your-api-key-here',`
   - 替換為：`OPENAI_API_KEY: 'sk-proj-xxxxx...',`（貼上您的 API Key）
   - 保存文件

### 第 2 步：打開應用

**方法 A：直接打開**
- 雙擊 `index.html` 文件
- 使用瀏覽器打開

**方法 B：使用本地伺服器（推薦）**
```bash
# 方式 1：使用 Python
python -m http.server 8000

# 方式 2：使用 Node.js
npx http-server

# 方式 3：使用 VS Code 的 Live Server 擴充
# 右鍵點擊 index.html -> Open with Live Server
```

然後在瀏覽器訪問：`http://localhost:8000`

### 第 3 步：開始分析

1. **選擇市場**：美股 / 台股 / 港股
2. **輸入股票代碼**：
   - 美股：`AAPL`、`TSLA`、`MSFT`
   - 台股：`2330`、`2317`、`2454`
   - 港股：`00700`、`09988`
3. **點擊「開始分析」**
4. **等待 30 秒 - 2 分鐘**（視分析深度而定）
5. **查看結果**

## 💡 使用技巧

### 省錢技巧
1. **日常使用選 3 級**（平衡成本與品質）
2. **測試時先選 1-2 級**（快速且便宜）
3. **重要決策才用 5 級**（最詳細但較貴）

### 提升準確度
1. **全選四個分析師**（獲得最全面的視角）
2. **選擇較新的分析日期**（更貼近當前市場）
3. **查看詳細報告**（不要只看摘要）
4. **交叉驗證**（結合其他資訊來源）

### 效率提升
1. **批次分析多檔股票**（但注意成本累積）
2. **保存重要分析結果**（截圖或複製文字）
3. **定期分析追蹤標的**（觀察趨勢變化）

## ⚠️ 常見問題速查

### Q1: 顯示 "API 調用失敗"
**解決方案：**
- 檢查 `config.js` 中的 API Key 是否正確
- 確認 OpenAI 帳戶有餘額
- 檢查網路連接

### Q2: 分析太慢
**解決方案：**
- 降低研究深度（改為 1-2 級）
- 減少選擇的分析師數量（只選 2-3 個）

### Q3: 分析結果看起來不對
**可能原因：**
- ChatGPT 的知識有截止日期（不包含最新消息）
- 股票代碼輸入錯誤
- 需要增加分析深度

### Q4: 費用會很高嗎？
**費用參考：**
- 單次標準分析：約 $0.05 - $0.10 USD
- 每天分析 5 檔：約 $0.25 - $0.50 USD
- 每月分析 100 次：約 $5 - $10 USD

**成本控制：**
在 OpenAI 平台設置每月使用上限

### Q5: 安全嗎？
**注意事項：**
- ✅ 您的 API Key 只存在本地電腦
- ✅ 分析請求直接發送到 OpenAI
- ❌ 不要將 config.js 上傳到公開網站
- ❌ 不要與他人分享 API Key
